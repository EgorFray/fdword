# Docxfixer

Docxfixer is a web application for formatting simple Microsoft Word (.docx) documents.

## Features

At the current version, you can format the following document properties:

- Font size
- Font type
- Line spacing
- Page margins
- First-line indent
- Text alignment (justify content)

You can also separately format the first paragraph (heading):

- Text alignment
- First-line indent
- Bold
- Capitalization

## Important

Current limitations of v1:

- Lists are not fully supported
- Tables are not fully supported

Support for lists and tables is planned for future updates.

## Planned Features (v2)

The next major update is planned to include document templates.

Users will be able to:

- Choose templates with multiple headings
- Configure each heading separately
- Apply different formatting rules to different sections of the document
